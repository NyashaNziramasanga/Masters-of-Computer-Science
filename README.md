## Setup Lecture Notes Script

![setup-lecture-notes.gif](Assets/images/setup-lecture-notes.gif)

1. In root cd folder name (semester):

```bash
cd 2020-semester-1
```

2. Run script:

```bash
node setupLectureNotes.js
```

3. Select course folder
4. Enter week number
5. Enter week title
